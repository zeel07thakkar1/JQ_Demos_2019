height()/width() : set or return height/width of an element(exclude padding , border , margin)

innerHeight()/innerWidth() : set or return height/width of an element(include padding)(give padding : 20px)

outerHeight() / outerWidth() method returns the width of an element (includes padding and border).(give padding : 20px ; border : 2px)

outerHeight(true) / outerWidth(true) method returns the width of an element (includes padding, border,margin).(give padding : 20px ; border : 2px ; margin : 10px)
